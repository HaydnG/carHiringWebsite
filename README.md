# carHiringWebsite
A professional car hiring website with Backend in Golang and Mysql with Frontend in Angular

Preview Images can be found at the front-end Git

# Features
- Sessions
- Cacheing
- Web Scraping
- Mysql
- CSV data loading
- .accdb data Loading
- Fully functioning car rental system
